# clone-instagram
노을, 조예리, 김예지 간단한 instagram 클론 코딩 프로젝트

## Github 사용법

### Clone
```
git clone https://github.com/ckyeon/clone-instagram.git
```

### Change Branch
```
git checkout -b [브랜치명]
```
- 이미 생성되어 있는 브랜치인 경우 `git switch [브랜치명]`

### Add
```
git add [staging 영역에 올릴 파일1] [staging 영역에 올릴 파일2]
```
- 전체 파일을 올릴 경우 `git add .`

### Commit
```
git commit -m "커밋 메시지"
```

### Push
```
git push origin [브랜치명]
```
- --set-upstream 설정이 되어있는 경우 `git push`
